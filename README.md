# Task Management App


A full-stack task management application built with authentication and database integration.

## Features

- User Authentication (Register/Login/Logout)
- Secure JWT-based authentication
- Create Tasks
- Update Tasks
- Delete Tasks
- User-specific task management
- Tasks saved in database
- User data saved in database
- Protected routes
- Full CRUD operations

## Tech Stack

- Frontend: React / Vite
- Backend: Node.js / Express
- Database: MongoDB
- Authentication: JWT
